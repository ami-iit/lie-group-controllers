13 Header-only C++ libraries containing controllers designed for Lie Groups.

15 ## Some theory behind the library
16 
17 The aim of the library is to contain some controllers designed in lie groups. The library depends only on `Eigen` and [`manif`](https://github.com/artivis/manif).
18 
19 All the controllers defined in `lie-group-controllers` have in common that they inherit from a templated base class ([CRTP](https://en.wikipedia.org/wiki/Curiously_recurring_template_pattern)). It allows one to write generic code abstracting the controller details. This follows the structure of `manif` and `Eigen`.
20 
21 The library implements two controllers:
22 1. Proportional Controller (`P controller`)
23 2. Proportional Derivative Controller (`PD controller`)
24 
25 The controllers have the following form
26 
27 | Proportional Controller | Proportional Derivative Controller |
28 |:-----------------------:|:-------------:|
29 | ![img-f6214bd2482f678b](https://user-images.githubusercontent.com/16744101/89174620-77c5b100-d586-11ea-88f7-318343c13b0f.png) | ![img-40c85670ed9bec65](https://user-images.githubusercontent.com/16744101/89174628-7b593800-d586-11ea-8219-d3ea2cb70901.png) |
30 
31 where `X` and `Xᵈ` are elements of a Lie group. `∘` is the group operator. `ψ` represents an element in the Lie algebra of the Lie group whose coordinates are expressed in `ℝⁿ`.
32 
33 At the moment, the controllers support all the group defined in `manif`. Namely:
34 - ℝ(n): Euclidean space with addition.
35 - SO(2): rotations in the plane.
36 - SE(2): rigid motion (rotation and translation) in the plane.
37 - SO(3): rotations in 3D space.
38 - SE(3): rigid motion (rotation and translation) in 3D space.

55 ## Build the library
56 
57 ```sh
58 git clone https://github.com/GiulioRomualdi/lie-group-controllers.git
59 cd lie-group-controllers
60 mkdir build && cd build
61 cmake ../
62 cmake --build .
63 [sudo] make install
64 ```
65 If you want to enable tests set the `BUILD_TESTING` option to `ON`.
66 
67 ## Use lie-group-controllers in your project
68 
69 **lie-group-controllers** provides native CMake support which allows the library to be easily used in CMake projects. Please add in your `CMakeLists.txt`
70 
71 ```cmake
72 project(foo)
73 find_package(LieGroupControllers REQUIRED)
74 add_executable(${PROJECT_NAME} src/foo.cpp)
75 target_link_libraries(${PROJECT_NAME} LieGroupControllers::LieGroupControllers)
76 ```
